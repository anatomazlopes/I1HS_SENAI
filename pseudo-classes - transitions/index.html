Exemplos de pseudo-classes em transitions

:hover

:hover aplica estilos quando o mouse está sobre o elemento.

:focus

:focus aplica estilos quando o elemento está em foco, como em um campo de entrada.

:active

:active aplica estilos enquanto o elemento está sendo clicado.

:visited

Link Visitado

:visited aplica estilos a links que já foram visitados pelo usuário.
