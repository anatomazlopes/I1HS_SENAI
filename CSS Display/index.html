Descrição dos Exemplos

1. display: block;
Elementos com display: block; ocupam toda a largura disponível e começam em uma nova linha. Exemplos incluem as tags <div> e <p>.

2. display: inline;
Elementos com display: inline; não iniciam uma nova linha e ocupam apenas o espaço necessário para seu conteúdo. Exemplos incluem as tags <span> e <a>.

3. display: inline-block;
Elementos com display: inline-block; combinam características dos elementos de bloco e inline. Eles podem ter largura e altura definidas, mas não iniciam uma nova linha. Um exemplo é a tag <img>.

4. display: none;
Elementos com display: none; não são exibidos na página e não ocupam espaço. Eles são completamente removidos do fluxo do documento.
